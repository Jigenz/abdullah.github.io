SecurityZAM - OSINT Mobile Application

Author: Abdullah Azzam Bin Syafrizal

Course: SKJ4333 - Mobile Security System

Overview: SecurityZAM is an OSINT mobile application designed to enhance security with features like secure authentication, two-factor authentication (2FA), geolocation services, and more.

Key Features

  • Secure User Authentication: Firebase authentication, strong password policies.
  • Two-Factor Authentication (2FA): Phone and email verification for enhanced security.
  • Geolocation Services: IP geolocation, GPS tracking.
  • QR Code Safety Analysis: Scans and verifies URLs using Google Safe Browsing API.
  • Runtime Application Self-Protection (RASP): Monitors for tampering and injection attacks.
  • Device Root Detection: Identifies compromised devices.
  • Risk-Based Scoring: Risk assessments based on device status and location.

Challenge Name: Download The Flag

Category: Miscellaneous (Misc)

Points: 500

Difficulty: Medium

Creator: SKRCTF

Challenge Overview

The objective was to retrieve the flag from a web server without downloading the entire 100 GB file. The provided information for the instance was:

IP Address: 52.148.100.67
Port: 30632
Lan Domain: 72-d5af8d6d-b3ea-4a35-b1b9-b0862e449658

Remaining Time: 3576s (~59 minutes)

Solution Steps

  1. Initial Reconnaissance:
    • Port Scanning: We used Nmap to identify services on the target IP and port:
      • nmap -sV 52.148.100.67 -p 30632
      PORT      STATE SERVICE VERSION
      30632/tcp open  http    nginx 1.27.2
    • Accessing the Web Server: Using curl to access the index page:
      • curl -v http://52.148.100.67:30632
      Output: Index of /

      The index page listed a 100 GB flag.txt file.

  2. Strategy Development:
    • Partial Content Retrieval using HTTP range requests.
    • Pattern searching for the flag in retrieved file chunks.
  3. Partial Content Retrieval:
    • Fetching the first 1,000 bytes:
      • curl -r 0-999 http://52.148.100.67:30632/flag.txt -o flag_start.txt
